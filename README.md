## Основные моменты

- Построение графа через `reachability_graph.construct_reachability_graph()`
- **Стандартная проверка Woflan**
- **Дополнительный анализ ограниченности** через граф достижимости
- **Анализ живости** через сильносвязные компоненты (SCC)
- **Проверка теоремы**: ограниченность + живость = бездефектность


- Генерация 50 трасс через `simulator.apply()`


- **Alpha Miner**: базовый алгоритм
- **Alpha+ Miner**: улучшенная версия
- **Heuristics Miner**: эвристический подход
- **Inductive Miner**: дерево процессов
- **DFG анализ**: графы непосредственного следования


- **Token-based Replay** для всех алгоритмов
- **Метрика Fitness**: average_trace_fitness, percentage_of_fitting_traces
- **Сравнительный анализ** 3 алгоритмов

- Проверка условий живости через SCC
- Построение конденсации графа
